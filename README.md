# Выпускная квалификационная работа (ВКР)

- **Название работы:** Разработка конвейера данных по мониторингу валютных пар (Development of a Data Pipeline for Monitoring Currency Pairs)
- **Студент:** Клаузер Андрей Сергеевич
- **Формат работы:** индивидуальная
- **Научный руководитель:** Паточенко Евгений Анатольевич
- **Научный соруководитель:** Заигрин Вадим Валерьевич
- **Описание проекта:** Целью выпускной квалификационной работы является разработка конвейера данных для мониторинга валютных пар, включающего сбор, обработку, анализ и визуализацию данных, а также построение и оценку моделей для анализа валютных курсов. Проект ориентирован на практическое применение методов работы с данными и машинного обучения в финансовой предметной области.

## Этап 3. Данные.
* **Описание:** Датасет содержит исторические суточные данные OHLCV (Open, High, Low, Close, Volume) по валютным парам EUR/USD, GBP/USD, USD/JPY, USD/CHF, AUD/USD, USD/CAD, NZD/USD, EUR/GBP, USD/RUB, EUR/RUB.
* **Источник:** Получено с помощью библиотеки [yfinance](https://pypi.org/project/yfinance/).
* **Использование:** Данные применяются для анализа доходностей, волатильности, корреляций и построения прогнозных моделей.
* **Загрузка данных и EDA:** Выгружены в ноутбук [baseline.ipynb](https://github.com/Andrey-Klauzer/vkr/blob/main/baseline.ipynb)

## Этап 4. Бейзлайн, метрики.

 ## Верхнеуровневый план работ.

1. Обзор источников:
   - Анализ научных публикаций и учебных материалов по теме обработки финансовых временных рядов  
   - Изучение подходов к построению data pipeline и ETL-процессов  

2. Обзор конкурентов и существующих решений:
   - Анализ существующих сервисов и платформ мониторинга валютных пар  
   - Сравнение архитектурных решений и используемых технологий  

3. Работа с данными:
   - Сбор данных по валютным парам из открытых источников (API, исторические данные)
   - Парсинг и очистка данных
   - Формирование и хранение датасетов  

4. Исследовательский анализ данных:
   - Анализ распределений и временной динамики валютных курсов  
   - Выявление трендов, сезонности и аномалий  

5. Построение бейзлайн-модели:
   - Выбор базовой модели прогнозирования
   - Определение и расчет метрик качества

6. Обучение продвинутой модели:
   - Подбор и обучение более сложных моделей
   - Сравнение результатов с бейзлайн-моделью
  
7. Построение конвеера:
  - Разработка интерактивного дашборда для мониторинга валютных пар  
  - Визуализация ключевых показателей и динамики курсов  

8. Создание сервиса:
   - Разработка сервиса для автоматизированного обновления и анализа данных
   - Интеграция моделей и визуализации  

9. Тестирование и доработка сервиса:
   - Проверка корректности работы конвейера данных
   - Оптимизация и улучшение производительности
